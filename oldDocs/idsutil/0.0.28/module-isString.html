Module: isString

判断对象是不是一个字符
Parameters:
Name Type Description
str object 目标对象
isEmpty boolean 是否开启空判断，true 是开启，false禁用（默认）
Author:
  • 王磊
Source:
  • index.ts, line 928
Returns:
true 代表是字符串，false 代表不是字符串
Type
boolean
Example
import wlutils from 'idsutil';
         const strT = "1";
         const emptyStrT = "";
         const intT = 1;
         // 断言是字符串
         wlutils.isString(strT);
         // 断言是字符串，但是由于字符串为空，所以返回false
         wlutils.isString(emptyStrT, true);
         // 断言是字符串，由于形参是number，返回false
         wlutils.isString(intT);
