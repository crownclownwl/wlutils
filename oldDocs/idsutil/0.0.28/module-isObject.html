Module: isObject

判断对象是不是一个object
Parameters:
Name Type Description
obj object 目标对象
isEmpty boolean 是否开启空判断，true 是开启，false禁用（默认）
Author:
  • 王磊
Source:
  • index.ts, line 858
Returns:
true 代表是object，false 代表不是object
Type
boolean
Example
import wlutils from 'idsutil';
         const array = [];
         const obj = {};
         // 断言是对象，由于形成是数组，返回false
         wlutils.isObject(array);
         // 断言是对象，返回true
         wlutils.isObject(obj);
         // 断言是对象，但是由于对象为空，返回false
         wlutils.isObject(obj, true);
